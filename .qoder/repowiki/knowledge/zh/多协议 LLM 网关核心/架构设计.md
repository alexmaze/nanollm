- **入口与运行时**：`server.ts` 作为核心入口，初始化 `ConfigManager` 并启动 Hono 服务；`cli.ts` 提供 CLI 启动封装。
- **配置管理**：`src/config-manager.ts` 实现基于文件监听（`watchFile`）的配置热更新机制，通过 `applyText` 方法原子化更新配置快照。
- **请求代理层**：`src/proxy.ts` 负责上游请求转发，包含 `passthroughRequest`（直通模式）和 `forwardRequest`（转换模式），并处理 TTFB 超时与流式内容校验。
- **协议转换层**：`src/converters/` 目录实现标准化中间表示（Normalized Request/Response），支持 OpenAI Chat/Responses 与 Anthropic Messages 之间的双向转换及 SSE 流式处理。
- **辅助模块**：`src/auth.ts` 处理 Bearer Token 鉴权；`src/fallback.ts` 维护失败计数器以支持动态故障转移排序；`src/status.ts` 与 `src/record.ts` 分别负责监控指标存储与请求录制。
